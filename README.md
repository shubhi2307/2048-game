# 2048 Game

A web-based implementation of the classic 2048 puzzle game developed using HTML, CSS, and JavaScript. The game allows users to combine numbered tiles to reach the 2048 tile while tracking scores and providing an interactive gaming experience.

## Features

- Interactive 2048 gameplay
- Tile merging and movement logic
- Real-time score tracking
- Responsive user interface
- Smooth user experience

## Technologies Used
- AWS Elastic Beanstalk (Deployment)

## Deployment

The application was deployed on AWS Elastic Beanstalk for hosting and management.
